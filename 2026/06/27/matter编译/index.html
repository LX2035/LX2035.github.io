5、参数说明

总结一下具体的参数如下

参数名称 默认值 说明
Setup PIN Code 20202021 配对时输入的 8 位数字密码
Discriminator 3840 用于发现设备的标识符
Manual Pairing Code 34970112332 手动输入的长串配对码
QR Code Content MT:-24J0AFN00KA0648G00 二维码包含的内容

其中MT:-24J0AFN00KA0648G00是最终呈现给用户的，它由下面这些信息组成，这个过程经历了一下计算

数据字段 位数 (Bits) 作用
版本 (Version) 3 Matter 规范版本号。
保留 (Reserved) 5 供未来使用。
配对 PIN 码 (Setup Passcode) 27 用于 PASE（Passcode-Authenticated Session Establishment）过程的 Setup PIN Code。在您的例子中，这个 PIN 码是 20202021。
鉴别器 (Discriminator) 12 12 位的值，用于在配网过程中，当网络上有多个设备在广播时，帮助控制器（Commissioner）快速识别并定位目标设备。
能力位掩码 (Discovery Capabilities) 8 8 位掩码，指示设备支持的配网通道： - Wi-Fi - Thread - 蓝牙低功耗 (BLE)
厂商 ID (Vendor ID - VID) 16 设备的生产厂商 ID。
产品 ID (Product ID - PID) 16 设备的具体产品型号 ID。
